Lagmite Language

A symbolic, expression-based programming language focused on strict typing and explicit operations.

Quick Example

[=]-[a][10]
[=]-[b][20]

[/DSP/]-[${[+]-[a][b]}]   → 30

Core Principles

  • No implicit type conversion
  • Strict operator-based syntax
  • Explicit evaluation using ${...}
  • Deterministic execution

Documentation Structure

  • Arithmetic → +, -, *, /, %, ^, R
  • Boolean → logical operations
  • Comparison → relational operators
  • System → assignment, display, conversion
