Boolean Operators

Overview

Boolean operators operate exclusively on BOO values and return BOO.

Type Signature

(BOO, BOO) → BOO
(BOO) → BOO   // unary only for NOT
---

[&] AND

Logical conjunction.

[&]-[TRUE][TRUE] → TRUE
[&]-[TRUE][FALSE] → FALSE
---

[!&] NAND

Negation of AND.

[!&]-[TRUE][TRUE] → FALSE
---

[!+] NOR

Negation of OR.

[!+]-[FALSE][FALSE] → TRUE
---

[X+] XOR

Exclusive OR.

[X+]-[TRUE][FALSE] → TRUE
[X+]-[TRUE][TRUE] → FALSE
---

[X!+] XNOR

Negation of XOR.

[X!+]-[TRUE][TRUE] → TRUE
---

[!] NOT

Unary logical negation.

[!]-[TRUE] → FALSE
---

Semantics

  • All operands must be of type BOO
  • No implicit type conversion occurs
  • Evaluation is strict
