Comparison Operators

Overview

Comparison operators evaluate relationships between values and return BOO.

---

Numeric Comparison

Type Signature

(NUM, NUM) → BOO
NUM ∈ {INT, DEC}

Operators

[>] Greater Than

[>]-[5][3] → TRUE

[<] Less Than

[<]-[3][5] → TRUE

[>=] Greater or Equal

[>=]-[5][5] → TRUE

[<=] Less or Equal

[<=]-[3][5] → TRUE
---

Equality Comparison

Type Signature

(T, T) → BOO

[==] Equality

[==]-[5][5] → TRUE

[!=] Inequality

[!=]-[5][3] → TRUE
---

Semantics

  • Numeric comparisons require INT or DEC
  • Equality requires matching types
  • If types differ:
    • == → FALSE
    • != → TRUE
