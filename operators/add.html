[+] Addition

Definition

Performs numeric addition, string concatenation, or boolean OR.

Type Signatures

(INT, INT) → INT
(DEC, DEC) → DEC
(INT, DEC) → DEC
(STR, STR) → STR
(BOO, BOO) → BOO

Semantics

Numeric values are added. Strings are concatenated. Booleans perform logical OR.

Examples

[+]-[5][3] → 8
[+]-["A"]["B"] → "AB"
[+]-[TRUE][FALSE] → TRUE
