[/DSP/] Display

Definition

Evaluates an expression and outputs its value to the console.

Syntax

[/DSP/]-[expression]

Semantics

  • The expression is evaluated
  • The result is converted to a printable form
  • The value is written to standard output
  • No value is returned

Output Formatting

INT

Printed as integer

DEC

  • Trailing zeros are removed
  • If value is integral → shown with .0

STR

Printed without quotes

BOO

TRUE or FALSE

Examples

[/DSP/]-[10] → 10
[/DSP/]-[10.500] → 10.5
[/DSP/]-[TRUE] → TRUE
[/DSP/]-["Hello"] → Hello

[/DSP/]-[${[+]-[5][5]}] → 10

Notes

  • Display does not modify program state
  • Errors during evaluation stop execution
